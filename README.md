# Proyectos de Programación en Python

¡Bienvenido a mi portafolio de proyectos prácticos en Python!

Este repositorio contiene pequeños programas de consola desarrollados con **Python puro** (sin librerías externas) para practicar y demostrar habilidades de programación como:
- Estructuras de datos (listas, diccionarios)
- Manejo de entrada/salida y archivos
- Control de flujo y manejo decycles
- Buenas prácticas de código limpio y organizado
- Manejo de exepciones
- Creacion de funciones

¡El repositorio está en crecimiento! Iré agregando nuevos proyectos con el tiempo.

## Proyectos incluidos

### 1. Gestor de Notas de Alumnos (`gestor_notas.py`)
Un sistema para registrar alumnos, sus materias, notas de exámenes, calcular promedios y generar un reporte detallado en archivo `.txt`.

**Características:**
- Agregar, mostrar y eliminar alumnos
- Cálculo automático de promedios
- Exportación a archivo de texto formateado

### 2. Gestor de Inventario (`gestor_inventario.py`)
Un gestor simple de stock de productos que permite agregar, eliminar, ver productos y generar un archivo `.csv` con valores totales.

**Características:**
- Manejo de productos (nombre, precio, cantidad)
- Cálculo de valor total del inventario
- Exportación a CSV compatible con Excel
- Formato de moneda regional en precios (punto para miles, coma para decimales)

### 3. Adivina el Número (`adivina_el_numero.py`)
Un juego clásico de adivinar un número aleatorio entre 1 y 100 con límite de intentos configurable y opción de jugar varias partidas.

**Características:**
- Pistas ("más alto/más bajo")
- Conteo de intentos
- Posibilidad de jugar de nuevo

## Cómo ejecutar los proyectos
1. Asegúrate de tener Python instalado (versión 3.6 o superior).
2. Clona o descarga el repositorio.
3. Abre una terminal en la carpeta del proyecto.
4. Ejecuta: `python nombre_del_archivo.py`

Ejemplo:
```bash
py nombre_del_archivo.py          # Windows 
python3 nombre_del_archivo.py     # Linux/macOS
